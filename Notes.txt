1. PROJECT DESCRIPTION
KrishiOS (Agri-OS) is an AI-powered agricultural platform that helps farmers with crop planning, market prices, disease detection, weather updates, and financial support.

2. REQUIREMENTS
- Modern browser (Chrome/Edge/Firefox)
- Internet (optional for AI features)
- 2GB+ RAM
- OpenRouter API key (optional)

3. HOW TO RUN PROJECT
1. Download project files
2. Open index.html in browser
OR run:
   python -m http.server 8000
3. Open localhost in browser

4. HOW IT WORKS (TECH FLOW)
User Input → Data Processing → AI Analysis → Local Storage/API → Results Display

5. FEATURES
- Crop planning
- Disease detection
- Market price prediction
- Weather insights
- Farmer marketplace
- Offline support (PWA)

6. COMMON ISSUES & FIXES
- Blank page → Refresh/check files
- AI error → Verify API key
- Slow app → Clear cache
- Data not saving → Enable LocalStorage

7. SUPPORTED BROWSERS
- Google Chrome
- Mozilla Firefox
- Microsoft Edge
- Safari

8. TIPS FOR BEST RESULTS
- Use updated browser
- Enter accurate crop data
- Backup data regularly
- Use internet for AI features
